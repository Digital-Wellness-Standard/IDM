# Design Manipulation Index (IDM)

The **Design Manipulation Index (IDM)** is an open framework developed by the **Digital Wellness Standard (DWS)** organization.  
It provides a structured methodology to identify, measure, and score manipulative or addictive design patterns in digital products ‚Äî promoting healthier, ethical user experiences.
